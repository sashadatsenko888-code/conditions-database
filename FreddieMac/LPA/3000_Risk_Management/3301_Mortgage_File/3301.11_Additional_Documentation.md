Source: Freddie Mac Seller/Servicer Guide  
Section: 3301.11 – Additional Documentation to Be Retained in Mortgage File  
Effective Date: 11/04/2020  

> **Condition-Relevant Section — Include in Ruleset**

---

## Overview
This section defines Freddie Mac’s requirements for **retaining all supplementary documentation** related to a Mortgage, including condominium, PUD, and cooperative loan materials, as well as taxpayer consent forms required under the **Taxpayer First Act**.  
These requirements ensure complete recordkeeping and compliance with Freddie Mac’s documentation standards.

---

## Requirements

1. **General Documentation Retention:**  
   - The Seller/Servicer must retain in the Mortgage file **all other Mortgage-related documents**, including:
     - Any documents required by the **Guide** or **Purchase Documents**,  
     - Any documents **requested by Freddie Mac**, and  
     - Any documentation **commonly maintained** in mortgage investor or servicer files.  
   - These materials must be **readily accessible** and **delivered at the Seller/Servicer’s expense** upon Freddie Mac’s request.

---

2. **Condominium, PUD, and Cooperative Loans:**  
   - For Mortgages secured by:
     - **Condominium Units**,  
     - **Planned Unit Development (PUD) units**, or  
     - **Cooperative Share Loans**,  
     the Mortgage file must include additional project-specific documentation required by Freddie Mac. Examples include:
     - **Declaration of condominium**,  
     - **Bylaws and regulations**, and  
     - **Cooperative project agreements** (per **Section 5705.11(c)**).  
   - Unless otherwise required by the Seller’s Purchase Documents, **project-level documentation** (e.g., master association materials) does **not** need to be maintained within each individual Mortgage file.

---

3. **Taxpayer First Act Compliance:**  
   - Under the **Taxpayer First Act**, a **signed taxpayer consent form** must be obtained **before disclosing** any taxpayer’s tax return or tax return information to a third party.  
   - If such consent is required, the Seller/Servicer must:
     - Obtain the **signed consent form in a timely manner**, and  
     - Place it in the **Mortgage file**.

   - For Servicers:
     - A **copy of the signed consent form** must be retained in the Mortgage file for every Mortgage where the Servicer obtains **tax return information** on or after **December 28, 2019**,  
       as part of Servicing activity for a Mortgage **owned or securitized by Freddie Mac**.

---

### Documentation Summary

| Loan Type | Required Additional Documentation |
|------------|----------------------------------|
| Condominium | Declaration, bylaws, regulations |
| PUD | Governing documents and applicable covenants |
| Cooperative Share Loan | Cooperative agreements, per Section 5705.11(c) |
| Any loan requiring taxpayer data | Signed taxpayer consent form (Taxpayer First Act) |
