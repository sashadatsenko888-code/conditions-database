## Overview
This section specifies Freddie Mac’s documentation and retention requirements for **Security Instruments** within the Mortgage file.  
It defines acceptable formats for recorded Security Instruments, including both paper and electronic records, and references applicable compliance sections for electronic recording and Manufactured Home Mortgages.

---

## Requirements

1. **Paper-Recorded Security Instruments:**  
   For Security Instruments recorded in paper format, the Mortgage file must contain **one** of the following:
   - The **original paper Security Instrument** with the official recordation notation, **or**
   - A **paper or electronic copy** of the Security Instrument that includes:
     - Recordation notation on the copy itself, **or**
     - A separate document maintained with the copy containing the recordation notation.

2. **Electronic Copies:**  
   If the Seller/Servicer maintains an **electronic copy** of the Security Instrument, they must comply with all requirements outlined in **Section 3302.2(b)** concerning electronic document storage, retention, and integrity.

3. **Electronically Recorded Instruments:**  
   For Security Instruments that were **electronically recorded**, the Seller/Servicer must comply with:
   - **Section 1401.14** — Electronic Recording of paper Closing Documents.  
   - **Section 1401.15** — Electronic Recording of Electronic Closing Documents.

4. **Manufactured Homes:**  
   For Mortgages secured by a **Manufactured Home**, refer to:
   - **Section 5703.4(b)** — Recording and title requirements.  
   - **Section 5703.4(f)** — Retention and documentation of title evidence.
